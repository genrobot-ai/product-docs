## 1.4 环境要求

### 1.4.1 操作条件

- 工作温度: **0°C–40°C**
- 存储温度: **-20°C–60°C**
- 湿度: **10–90% (non-condensing)**

### 1.4.2 环境注意事项

- 避免强磁场和长时间阳光直射.

# 2. 产品概述

## 2.1 产品介绍

**DAS（数据采集系统）** 是一个通用的具身智能多模态数据采集系统. **DAS 夹爪** 是 DAS 系列中的一款双指数据采集设备。基于先进的 UMI 设计理念，它集成了五种类型（包含八个单元）的传感器，即视觉、触觉、听觉、惯性以及磁编码传感器，能够高效采集具身智能多模态数据。.

![](https://cdn.shopify.com/s/files/1/0777/8874/1847/files/MK_2_renders.82.jpg?v=1762741580)

## 2.2 核心特性

- **便携易用**: 轻量化设计提供卓越的操控性，达到媲美人工的操作效率.
- **多模态数据采集**: 集成了仿生复眼、IMU、触觉、麦克风、磁编码器等多模态传感器
- **多末端执行器，通用适配性**: 提供多种末端执行器，用于处理不同形状、尺寸和重量的物体.
- **本地/云端双模式**: 本地数据处理确保数据安全，云端模式便于统一管理和远程采集.
- **本体无关性**: 可部署于多种机器人平台.
- **产品级定位**: 开箱即用，用户友好，提供便利且易于使用.
- **顶级工业设计**: 由劳斯莱斯设计师设计.

## 4. 数据采集快速入门指南

### 步骤 1：开箱检查

1. 打开包装盒，取出所有组件
2. 组装夹爪头.![](https://cdn.shopify.com/s/files/1/0777/8874/1847/files/AssembleGripperHead.jpg?v=1763551432)
   - 通用夹爪: 适用于大多数抓取任务.
   - 触觉夹爪: 适用于需要精确力控的任务.
   - 软体夹爪：提供更好的贴合性.
3. 插入 MicroSD 卡，按一次电源按钮启动设备，LED 指示灯将发出绿光。当前启动过程可能比平时长。我们感谢您的耐心，并正优先优化后续更新。如果启动时间超过 30 秒，请联系技术支持.
4. 第一次使用 SD 卡时，需要格式化 SD 卡：下拉屏幕 - 点开设置 - 找到 Format Micro SD Card - 点击 Confirm，系统将格式化该 SD 卡。格式化一次后，后续可正常使用，若需要清空数据，也可使用格式化功能

### 步骤 2：开始采集

1. 点击录制按钮开始数据采集
2. 屏幕上出现录制计时
3. 保持设备静止两秒后，执行您想要采集的任务动作，按下触发器控制夹爪
4. 再次点击录制按钮停止数据采集
5. 录制计时消失，数据自动保存到 MicroSD 卡

**注意事项**

1. 采集时倾斜角度不要太大，尽量水平，朝着有一定纹理的区域开始采集
2. 点击录制按钮停止采集时，点击按钮最慢 2s 内数据保存完毕，请耐心等待

### 步骤 3：数据传输

#### MicroSD 卡读取（推荐）

1. **关闭** 设备电源.
2. 轻按 MicroSD 卡，将其插入读卡器，并将读卡器连接到电脑![](https://cdn.shopify.com/s/files/1/0777/8874/1847/files/microSD_card_position.png?v=1763123203)
3. 打开 **Matrix Studio​** 并导入您的数据.

## 5. 数据采集注意事项

##### 设备检查：

- 电池电量 > 5%
- SD 卡可用空间 > 100MB
- 所有传感器工作正常

##### 采集质量提示

1. 保持动作自然流畅.
2. 避免遮挡摄像头视野.
3. 避免在非常黑暗的环境中使用
4. 避免快速剧烈晃动.
5. 避免使用纹理贫乏的背景，如白墙、空桌面.

## 6. 软件更新

### 6.1 固件更新流程

1. 将固件文件复制到 MicroSD 卡
2. 将卡插入设备以触发更新
3. 完成后系统将自动重启
4. 在移除 MicroSD 卡前关闭设备电源

### 6.2 UI 更新

前往 设置 > 版本检查 以检测并安装新更新.

## 7. 设备维护

### 日常清洁

#### 摄像头清洁

- 用干净的微纤维布轻轻擦拭镜头
- 避免使用化学清洁剂
- 定期检查镜头是否有划痕

#### 屏幕清洁

- 在设备关闭时清洁屏幕
- 使用专用的屏幕清洁剂
- 避免用力按压

#### 外壳清洁

- 用潮湿的软布擦拭外壳
- 防止湿气进入接口
- 清洁后确保完全干燥

### 存储管理

#### SD 卡维护

- 及时传输重要数据
- 避免在设备开机时拔出 SD 卡.
