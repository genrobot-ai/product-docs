# 双手采集数据指南V2.0

双手设备采集V2.0 主要通过无线频段配对进行数据采集

## 注意事项

**（1）在每次开机后，需要手动设置设备为左右夹爪，并点击确认配置按钮**  
**（2）当多个设备在一块的时候，确保每两个设备频段和同步字相同，避免串频**  
**（3）当确认配置按钮变为灰色后，回到主界面需查看页面上有左手或右手图标，配对成功即可进行录制**  
**（4）实际采集时，请保证开始采集时两台设备看向同一处位置，静止1s后再开始动作，保证初始化完成**  
**（5）如果两台设备一直没右连接上，建议切换频段和同步字，还是不行就重启一次再重新连接**


## 1.如何在界面上配对

**（1）长按打开两台设备的配对开关**  

<img src="https://cdn.shopify.com/s/files/1/0777/8874/1847/files/dual_hand_setiing.png?v=1770445671" alt="Setting1" width="300" height="200">

**（2）进入配对页面**  

<img src="https://cdn.shopify.com/s/files/1/0777/8874/1847/files/dual_hand_wifi.webp?v=1770445640" alt="Setting2" width="300" height="200">

**（3）以左手的设备作为主机，选择频段，同步字和左夹爪，并点击确认配置，等待图标变为灰色即可**  

**（4）长按右手设备的配对开关，选择与左手设备相同的频段和同步字，选择右夹爪，并点击确认配置，等待图标变为灰色即可**  

**（5）连接成功后两台设备的录制界面上方应分别显示左边和右边的配对标识**  

**（6）配对成功后，只有左手设备的录制按钮即可控制双手采集数据的录制和终止，右手设备只受左手控制**  

## 2. 录制数据到sd卡

在录制数据到SD卡后，bag name 将区分主从机：  
主机：DAS-Gripper_20260127193045_left_leader_00000000.mcap  
从机：DAS-Gripper_20260127193045_right_follower_00000000.mcap

## 4.双手采集注意事项
1. 采集前静止 3s 以上  
2. 采集开始和结束时,两个夹爪的左摄像头尽量看向同一处位置，重叠范围越大越好。共视物体在远处，更容易融合两个夹爪的坐标系  
3. 单包时长过长会影响数据处理速度，每一个任务的录制时长尽量控制在15-60秒  
4. 采集过程中避免长时间视野被桌面/动态物体遮挡  
5. 两设备的左摄像头视野内尽量纹理丰富，除被夹取物体外，有其他静态物体
