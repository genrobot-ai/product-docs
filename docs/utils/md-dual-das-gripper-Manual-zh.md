# 双手采集数据指南V1.0

双手采集V1.0 主要通过双端type-c USB线进行数据对齐

## 注意事项

**（1）在配对时应保证usb连接牢固，同时通过绑带，尽量减少USB口在采集时被拉扯，如录制界面的配对标识时隐时现则说明usb连接不稳（参考固定方案将由简智提供）**

**（2）配对成功后应通过左手设备的录制按钮进行录制操作**


## 1.如何在界面上配对

**（1）使用双头USB线连接两台设备**

**（2）打开两台设备的配对开关**  

<img src="https://cdn.shopify.com/s/files/1/0777/8874/1847/files/main.jpg?v=1766653127" alt="Setting1" width="300" height="200">

**（3）以左手的设备作为主机，长按左手设备的配对开关，在配对界面选择“左”**

**（4）然后长按右手设备的配对开关，在配对界面选择“右”**

**（5）连接成功后两台设备的录制界面上方应分别显示左边和右边的配对标识**

**（6）最后通过左手设备的录制按钮即可控制双手采集数据的录制和终止，右手不受控**

## 2. 录制数据到sd卡

在录制数据到SD卡后，bag name 将区分主从机：  
主机：host_eb208e5ece7fb883_20251226231342.mcap  
从机：host_eb208e5ece7fb883_device_445aa23a732f5815_20251226231342.mcap
