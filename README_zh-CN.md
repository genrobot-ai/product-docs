## 🚀 产品系列

### DAS (Data Acquisition System) 数据采集系统

**DAS** 是一款通用型具身智能多模态数据采集系统，专为机器人学习和研究设计。

#### DAS Gripper - 双指数据采集设备

DAS Gripper 是 DAS 系列中的双指数据采集设备，基于先进的 UMI 设计理念，集成了五大类（包含八个单元）传感器：

- 🎥 **视觉传感器** - 仿生复眼相机系统
- 🤏 **触觉传感器** - 高精度力反馈
- 🎤 **声学传感器** - 环境音频采集
- 🧭 **惯性传感器** - 六轴 IMU
- 🔄 **磁编码传感器** - 精确位置追踪

**核心特性：**

- 🎯 便携友好：轻量化设计，操作效率媲美人工
- 📊 多模态数据采集：集成多种传感器实现全方位数据获取
- 🔧 多样化末端执行器：适配不同形状、尺寸和重量的物体
- ☁️ 本地/云端双模式：确保数据安全的同时支持远程管理
- 🤖 硬件无关性：可部署于多种机器人平台
- 🎨 顶级工业设计：由劳斯莱斯设计师打造

### Matrix Studio - 数据处理与可视化平台

**Matrix Studio** 是配套的数据处理和可视化平台，提供完整的数据分析工具链：

- 📈 数据可视化：交互式图表和多视角数据展示
- 🔬 传感器标定：多源传感器参数校准
- 🎯 Ground Truth 生成：高精度轨迹恢复算法（VIO）
- 🗂️ 数据管理：MCAP 格式数据的导入、处理和导出
- 🌐 灵活部署：支持本地模式和 HTTPS 网络访问
